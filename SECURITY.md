# Security Policy

## Supported versions

| Version | Supported |
| ------- | --------- |
| 1.0.x   | Yes       |
| < 1.0   | No        |

## Reporting a vulnerability

If you discover a security vulnerability in DomainHunter, please **do not open a
public issue**. Instead, email `mohammad.pooshesh@gmail.com` with:

- a description of the issue and its impact,
- steps to reproduce, and
- any suggested remediation.

You can expect an acknowledgement within 72 hours and a status update within
seven days.

## Responsible use

DomainHunter is a reconnaissance tool intended for authorized security testing,
research and defensive purposes only. Only scan domains you own or have explicit
written permission to assess. Misuse may violate local, national or
international law. The authors accept no liability for misuse.
